Merge
-----
A merge combines the history of one branch with another.
After successful merging, all committed changes become
available in the target branch.

Useful Git Commands
-------------------

git checkout main
    Switches back to the main branch.

git merge GitNewBranch
    Merges the feature branch into main.

git branch -d GitNewBranch
    Deletes the branch after merging.

Best Practices
--------------

1. Commit frequently.
2. Use meaningful commit messages.
3. Merge only after testing.
4. Delete unused branches.
5. Keep the main branch stable.
